# Line Follower PID + Fuzzy

## Deskripsi
Line Follower PID + Fuzzy merupakan robot mobile berbasis Arduino Nano yang dirancang untuk mengikuti lintasan secara otomatis dengan menerapkan kombinasi algoritma PID (Proportional–Integral–Derivative) dan Fuzzy Logic. PID digunakan untuk mengurangi error posisi robot terhadap garis, sedangkan Fuzzy Logic berperan dalam menyesuaikan parameter kendali agar pergerakan robot menjadi lebih stabil, responsif, dan adaptif terhadap perubahan lintasan.

## Fitur
- Mengikuti lintasan secara otomatis (Line Following)
- Implementasi algoritma PID
- Implementasi Fuzzy Logic
- Pengaturan kecepatan motor secara adaptif
- Pembacaan sensor secara real-time
- Kendali motor berbasis PWM
- Respon stabil pada lintasan lurus maupun tikungan

## Teknologi

- PID Control
- Fuzzy Logic
